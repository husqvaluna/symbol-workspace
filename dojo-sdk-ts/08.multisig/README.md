# マルチシグ

あるアカウントが行うトランザクションの実行に、他のアカウントからの承認を必要とするアカウントを構築するトランザクションです。

一つのアカウントからの操作を複数のアカウントから実施できるようにしたり、複数の承認を要求できるように設定できます。


## トランザクションの作成

新しいアカウントを生成し、それをマルチシグアカウントとして設定します。

設定後はアカウント1とアカウント2からの承認があって、トランザクションを実行できる状態となります。

[multisig-modification.ts](./multisig-modification.ts)


## マルチシグアカウントからの転送

マルチシグ化したアカウントから、`XYM`を転送します。

そのトランザクションが承認されるためには、アカウント1とアカウント2の署名があって成立します。

[initiate-from-multisig-account.ts](./initiate-from-multisig-account.ts)
