# モザイク定義の送信

モザイクの定義を行う。
配布可能なモザイクを発行するには別のトランザクションの発行が必要だが、ここでは省略。
後述するアグリゲートトランザクションにて扱う。


## ディスクリプタオブジェクト作成

トランスファートランザクションの項でやったように、モザイク定義のトランザクションディスクリプタを作成する。

[new-mosaic.ts](./new-mosaic.ts)

なお、基本的にディスクリプタの構築がトランザクションごとに異なるだけなので、
以降のコードにはアカウントオブジェクト生成やトランザクション署名について同じコードが含まれる。

トランザクションのアナウンスや確認の方法も同様のため、省略する。
